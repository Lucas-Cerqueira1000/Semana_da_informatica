Sobre o App

     Este aplicativo foi desenvolvido como parte da Semana da Informática. Ele utiliza componentes como SeekBar, Spinner, Intent e mais. Esse aplicativo tem a funcionalidade de que ao selecionar um gênero cinematográfico aparece alguns filmes desse gênero, e ao clicar nesse filme irá aparecer a sinopse do filme.

        btnMostrar.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
                String genero = selecionar.getSelectedItem().toString();
                ArrayList listaSelecionada;

                switch (genero) {
                    case "Terror":
                        listaSelecionada = ListaTerror;
                        break;
                    case "Drama":
                        listaSelecionada = ListaDrama;
                        break;
                    case "Comédia":
                        listaSelecionada = ListaComedia;
                        break;
                    case "Romance":
                        listaSelecionada = ListaRomance;
                        break;
                    case "Ficção Científica":
                        listaSelecionada = ListaFiccao;
                        break;
                    case "Ação":
                        listaSelecionada = ListaAcao;
                        break;
                    case "Animação":
                        listaSelecionada = ListaAnimacao;
                        break;
                    default:
                        listaSelecionada = new ArrayList<>();
                }

                adapter = new FilmeAdapter(MainActivity.this, listaSelecionada);
                listinha.setAdapter(adapter);
            }
        });
    }

Curiosidades


  • As concorrentes da época em que o Sistema Operacional da Google ainda estava engatinhando não deram nenhum crédito para a ameaça que o robozinho verde representava;
  • Os nomes das versões do Android são inspirados em doces, seguindo a ordem alfabética;
  • O criador do sistema Andy Rubin havia concebido o Android para ser uma plataforma para equipar câmeras digitais;
  • O Android é baseado no kernel Linux e foi inicialmente criado pela Android Inc. em 2003.
